Boardwalk Games

Here are the key things you need for this challenge: Open your project in the recommended workspace (IDE) for this course. Create the correct folder structure to hold a folder for an index.html file, and assets such as images and css. Download and unzip this zip file of images. Then, add them to your workspace in the correct folder. Create the boilerplate HTML code for index.html using the !+TAB shorthand. Add some heading text and a paragraph to index.html and run the code to check it is working. Create a style.css file, ensuring the correct folder structure. Connect the HTML to style.css. When you have confirmed everything is working, commit your code. If you need help, the full steps can be found below. Please give this challenge a good try before checking the steps.
